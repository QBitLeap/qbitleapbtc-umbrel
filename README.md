# QBitLeap BTC for Umbrel

Makes the official Qbit `qbit-mining-bootstrap` project run as an Umbrel-native application with minimal upstream modifications.

## Goals

- Keep upstream code as unchanged as possible.
- Package the official Qbit mining bootstrap as an Umbrel app.
- Support Bitcoin + Qbit merge mining.
- Validate each milestone before adding new features.
- Prefer upstream implementations over custom code.

## Status

Early development.
